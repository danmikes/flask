## Configuration

### PythonAnyWhere
  - Virtual Environment : `python3.10 -m venv .venv`
  - Library : `python3.10 -m pip install -r conf/pack3.10.txt`
  - Proxy-Server : `uWSGI`
  - URL : `https://dmikes.eu.pythonanywhere.com`
  - Configuration : `wsgi.py`
